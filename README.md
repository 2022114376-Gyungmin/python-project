# 프로젝트 제안서

## 프로젝트 제목
**대구 NDVI 대시보드 - 위성영상 기반 산불 피해지 식생 변화 분석**

## 한 문장 설명
Sentinel-2 위성영상으로 대구광역시 산불 피해 지역의 식생 변화를 자동으로 분석하고 인터랙티브 웹 대시보드로 시각화하는 프로젝트

## 풀려는 문제 / 만들려는 이유
2025년 4월 말 대구 함지산 산불을 비롯해 최근에도 전국 곳곳에서 산불이 자주 발생하고 있다. 그러나 이러한 상황을 대부분 뉴스로 접할 뿐 그 피해가 얼마나 심각한지, 식생이 얼마나 회복되었는지 등 객관적으로 확인할 수 있는 도구가 부족하다. 유럽우주국 Sentinel-2 위성이 5일 간격으로 지구를 촬영해 무료로 데이터를 제공하고 있지만 비전문가가 활용하기에는 진입장벽이 높다고 생각된다. 따라서 전공지식을 적용해 위성영상 분석 결과를 누구나 클릭만으로 확인할 수 있는 도구를 만들어 보고 싶다는 생각을 하게 되었다.

## 핵심 기능 3가지
**1. 위성영상 자동 수집 및 NDVI 계산**
- Sentinel-2 영상에서 지정 시기·지역의 구름 없는 영상을 자동으로 가져와 식생지수(NDVI) 계산

**2. 산불 전후 비교 시각화**
- 산불 발생 전과 후 시점의 NDVI를 지도와 그래프로 동시에 비교

**3. 인터랙티브 웹 대시보드**
- 사용자가 분석 지역과 시기를 선택하면 결과가 실시간으로 지도에 표시됨

## 예상 사용자
1. 본인
2. 관련 학과 재학생들이나 위성영상 분석에 관심 있는 비전문가

## 결과물 형태
**웹 - Streamlit 기반 인터랙티브 대시보드**
- 브라우저에서 실행
- 클릭으로 분석 영역과 시기 선택
- 지도, 그래프, 통계 수치를 한 화면에서 확인

## 사용할 Python 개념
**변수(Variable) :** 좌표값, 시기, 분석 결과 저장

**자료구조**
- 리스트(list) : 밴드 목록, 좌표
- 딕셔너리(dict) : 시각화 옵션

**함수(Function) :** NDVI 계산, 영상 가져오기 등

**메소드 체이닝(Method Chaining) :** 데이터 처리 파이프라인 구성

**외부 라이브러리 사용**
- Google Earth Engine 위성영상 처리
- geemap : 지도 시각화
- streamlit : 웹 대시보드 구현
- pandas, matplotlib : 데이터 처리 및 그래프

**조건문 / 반복문 :** 여러 시기 영상 자동 처리
