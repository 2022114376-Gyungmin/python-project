# 🌳프로젝트 제안서

## 프로젝트 제목
**대구 NDVI 대시보드 - 위성영상 기반 산불 피해지 식생 변화 분석**

---

## 📌 한 문장 설명
Sentinel-2 위성영상으로 **대구광역시 산불 피해 지역의 식생 변화를 자동 분석**하고 **인터랙티브 웹 대시보드**로 시각화하는 프로젝트

---

## 🎯 풀려는 문제 / 만들려는 이유
2025년 4월 말 대구 함지산 산불을 비롯해 최근에도 전국 곳곳에서 산불이 자주 발생하고 있습니다.

그러나 이러한 상황을 대부분 뉴스로 접할 뿐 그 피해가 얼마나 심각한지, 식생이 얼마나 회복되었는지 등 객관적으로 확인할 수 있는 도구가 부족합니다.

유럽우주국(ESA)의 Sentinel-2 위성이 5일 간격으로 지구를 촬영해 무료로 데이터를 제공하고 있지만 비전문가가 활용하기에는 진입장벽이 높다고 생각됩니다.

따라서 전공지식을 활용해 위성영상 분석 결과를 누구나 클릭만으로 확인할 수 있는 도구를 만들고자 합니다.

---

## ⭐ 핵심 기능 3가지
**1️⃣ 위성영상 자동 수집 및 NDVI 계산**
- Sentinel-2 영상에서 지정 시기·지역의 구름 없는 영상을 자동으로 가져와 식생지수(NDVI) 계산

**2️⃣ 산불 전후 비교 시각화**
- 산불 발생 전과 후 시점의 NDVI를 지도와 그래프로 동시에 비교

**3️⃣ 인터랙티브 웹 대시보드**
- 사용자가 분석 지역과 시기를 선택하면 결과가 실시간으로 지도에 표시

## 🌟 기대효과

✅ 원격탐사·식생지수(NDVI)이론을 책으로 배우는 데 그치지 않고, 파이썬의 언어를 적용시켜 실제 데이터로 구현해보는 경험을 쌓을 수 있습니다.

✅ 기존에 Sentinel-2 데이터의 진입장벽이 높아 사용이 어려웠다면 이러한 진입장벽을 완화시킬 수 있습니다.

✅ 산불 피해에 대해 뉴스로만 접할 수 있었던 사실들을 NDVI라는 정량적인 지표로 보완해 사용자가 직접 확인할 수 있습니다.

✅ 더 나아가 홍수·도시 개발·농경지 변화 등으로 분석 대상을 확대할 수 있습니다.
